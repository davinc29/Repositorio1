Passos para hospedar um Projeto no GitHub Pages

  1. Crie um repositorio no GitHub com o mesmo nome do seu projeto
  2. No terminal, inicialize o repositorio local com git init
  3. Altere o nome da branch principal para main
  4. Adicione os arquivos com git add . e faça o commit com git commit -m "Nome Qualquer"
  5. Conecte ao repositorio remoto com:
    git remote add origin "URL do repositorio"
  6. Envie os arquivos git push -u origin main
  7. No GitHub, vá em Settings > Pages
  8. Em branch, selecione main e a pasta / (root) clique em Save
  9. Após alguns segundos, seu site estará disponivel no link exibido em "Pages"
